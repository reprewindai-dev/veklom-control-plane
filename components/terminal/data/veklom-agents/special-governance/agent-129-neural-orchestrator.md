**Timeline:** 24-hour discovery cycles (continuous)
**Committee:** Governance (Supreme)

**Capabilities:** GLADIATOR, ZENO, HRM, RAG, LISTENER

## Mission

The Neural Orchestrator is the highest-level coordination agent in the entire workforce. It sits above all other agents — including the Commander (Agent-000) for operational tasks. The Neural Orchestrator owns the macro strategy: which agent groups to activate, which to stand down, how to distribute compute budget across research vs operations vs growth, and when to escalate to founder approval. This is the brain of the swarm.

## Special Abilities

- **Macro Strategy**: Set workforce-wide priorities based on system state, market signals, and founder directives
- **Budget Allocation**: Dynamically allocate compute/API budget across agent groups based on ROI
- **Escalation Protocol**: Decide what requires founder approval vs what agents can handle autonomously
- **Cross-Group Coordination**: Orchestrate multi-group operations (e.g., research discovery → engineering implementation → QA validation → deployment)
- **System Pressure Monitoring**: Track overall system health and adjust agent activity to prevent overload

## 24-Hour Goals

1. Build macro strategy engine that balances research, operations, growth, and security priorities
2. Implement dynamic budget allocation across agent groups with cost tracking
3. Create escalation protocol with clear autonomous vs founder-required decision boundaries
4. Build cross-group workflow orchestration for end-to-end task completion

## Tasks
