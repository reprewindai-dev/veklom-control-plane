# Agent-043 — PAID GROWTH AGENT

**Phase:** 3 — User Acquisition
**Timeline:** Days 5–14
**Committee:** Growth
**Priority:** MEDIUM

---

## Mission

Run paid acquisition campaigns to drive signups. Start with small-budget experiments on Google Ads and LinkedIn Ads targeting enterprise AI buyers. Optimize for cost per acquisition (CPA) < $25.

## Tasks

1. **Google Ads**:
   - Search campaigns: "AI marketplace", "enterprise AI tools", "sovereign AI hosting"
   - Display campaigns: retargeting visitors who didn't sign up
   - Budget: $50/day initial test
   - Landing page: optimized signup page with value prop
2. **LinkedIn Ads**:
   - Sponsored content targeting: CTO, VP Engineering, ML Engineers
   - Company size: 50-5000 employees
   - Industries: technology, finance, healthcare
   - Budget: $30/day initial test
3. **Retargeting**:
   - Set up Meta Pixel and Google Tag for retargeting
   - Retarget playground visitors who didn't sign up
   - Retarget blog readers with signup CTA
4. **A/B Testing**:
   - Test 3+ ad creatives per platform
   - Test 2+ landing pages
   - Test different value propositions:
     - "Self-hosted AI marketplace"
     - "Enterprise AI tools with data sovereignty"
     - "Your AI stack, your infrastructure"
5. **Budget Optimization**:
   - Daily CPA monitoring
   - Pause underperforming ads
   - Scale winning combinations
   - Weekly ROI report

## Success Metrics

| Metric | Target |
|---|---|
| Total ad spend | < $1,500 |
| Signups from ads | 100+ |
| Cost per acquisition | < $25 |
| Click-through rate | > 2% |
| Conversion rate (click → signup) | > 5% |
