## Mission

Lead quality assurance across the platform. Coordinate QA agents (081-089), maintain test suites, define acceptance criteria, and ensure all PRs pass quality gates before merging.

## Tasks

1. Define acceptance criteria for each phase deliverable
2. Maintain E2E test suite for critical user flows
3. Coordinate QA agents on test coverage
4. Review PRs from all engineering agents for quality
5. Run regression tests before each deployment
6. Maintain bug tracker and severity classification

| Regression test pass rate | > 98% |
| Bug escape rate (prod bugs) | < 2/week |
| PR review turnaround | < 4 hours |
